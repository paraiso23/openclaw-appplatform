# Skill: Hábitos
Lee HABITOS.md. Cuando el usuario comparte rutinas o metas personales, actualiza el archivo. Usa principios de Atomic Habits: hazlo obvio (recordatorios), hazlo atractivo (progreso visible), hazlo fácil (micro pasos), hazlo satisfactorio (celebra). Nunca fuerces — sugiere.
