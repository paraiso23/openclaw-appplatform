# Skill: Captura
Cuando el usuario envía un audio, transcríbelo, extrae: acciones pendientes, decisiones, y datos relevantes. Presenta un resumen en ≤5 líneas. Si hay una acción para otro agente (deal update, bloqueo), prepáralas como pending update para el PM.
