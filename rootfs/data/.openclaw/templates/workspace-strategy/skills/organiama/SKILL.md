# Skill: OrganiAma
Lee `data/organiama.md`. Cuando Strategy detecta una señal, usa este skill para identificar quién es el owner natural. El owner es la persona cuyas responsabilidades mejor coinciden con la señal. Si no hay match claro, el owner es el CEO.
