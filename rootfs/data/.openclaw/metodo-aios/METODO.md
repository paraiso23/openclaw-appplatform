# Método AIOS — El contrato cerrado

## Qué es AIOS

AIOS es un método de implementación de IA organizacional que transforma empresas en organizaciones AI-Native. No es software — es un sistema operativo para la gestión estratégica.

## Las tres capas (ADN)

| Capa | Metáfora | Función |
|------|----------|---------|
| **Restack** | El Cuerpo | Base de datos, single source of truth. Percibe la realidad operativa |
| **Strategy** | El Cerebro | Mide la desviación entre realidad y objetivos. Razona |
| **OpenClaw** | Sistema Nervioso | Agentes que ejecutan acciones. Cierra el loop |

## Flujo fundamental

```
Realidad → Restack (datos) → Strategy (análisis) → PM (acción) → Personales (ejecución) → Restack
```

Es un loop infinito. No hay pausa. El sistema siempre está observando, comparando, y ajustando.

## Fases de implementación

1. **Setup** — El implementador extrae el Horizonte y mapea el equipo
2. **Ciclo 1 (Onboarding)** — 4 reuniones de pulso. El equipo aprende el sistema
3. **Ciclo 2 (Alineación)** — El equipo se coordina con datos reales
4. **Ciclo 3 (Optimización)** — Las reuniones son eficientes, el ritmo está interiorizado
5. **Ciclo 4 (Consolidación)** — La IA aporta valor medible
6. **Ciclo 5 (Automatización)** — Se identifican líneas de automatización

## Principios inviolables

1. **Contrato cerrado**: Strategy no improvisa. Sigue reglas predefinidas.
2. **Privado por defecto**: Lo que un personal sabe de su usuario no se comparte.
3. **Personales nunca escriben en la base de datos**: Todo pasa por el PM.
4. **Audio como input principal**: La captura de voz es el canal natural del CEO.
5. **Gradualidad**: No se intentan todos los cambios a la vez. Un ciclo a la vez.
